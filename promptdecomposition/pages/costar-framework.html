CO-STAR Framework

The Professional's Prompt Template

Overview

CO-STAR is a structured framework that ensures every prompt has the necessary context and constraints to be successful.

Breakdown

  • C (Context): Background information.
  • O (Objective): The specific task.
  • S (Style): The writing style.
  • T (Tone): The emotional quality.
  • A (Audience): Who is this for?
  • R (Response): Format constraints.

Example

Prompt: 
(Context) You are a social media manager.
(Objective) Write a tweet about a new product.
(Style) Minimalist.
(Tone) Exciting.
(Audience) Tech enthusiasts.
(Response) Under 280 characters.
